Welcome

theMettƒÅShelf is a repository for Buddhist Books, with a built in app for reading ebooks and listening to AudioBooks.

Currently this app is still in it's infancy and we consider it a prototype. There will be many design changes over the coming years as we try out functionality. If you have any ideas, please feel free to contact us, either via the contact form on the main site (see below) or you can message the librarian @stu on the SuttaCentral forum.

Instructions

To read a book, simply click on it.

To get back to this, the library page, click on the Library logo in the top left hand corner of any book.

For settings click the cog wheel in the top right. Once you are in a book, there are additional settings available.

All settings and your place in each book is stored on the device that you are using. No information is sent to Wisdom & Wonders.

You can search for a book on this page using the 'FIND' menu on the left hand side of this page.

Resources

There is a complete list of all ebook downloads on our main site, including non-English language books, not in this app.

If you have any problems, comments or suggestions for future versions, please visit our contacts page on our main site.

Help

Sutta Search Help:
  • Searches on the Sutta Number will return books that reference that sutta in their pages.
  • We use SuttaCentral style numbering. e.g. try an4.170 or mn66
  • We are currently indexing the books in the library where they reference sutta's. Only the shorter booklets have been indexed so far.
  • Hint: To see which books have currently been indexed, press the Sutta Number button and leave the search field blank.
  • As soon as you start your search, all books will disappear beacuse you must enter the exact number of the sutta that you are looking for. For example, an4.17 will not return a book where only an4.170 is referenced.
